Мониторинг загруженности столовой «Восточное»

Этот проект — веб-интерфейс для отображения загруженности столовой в режиме, близком к реальному времени.
Система рассчитана на интеграцию с backend и ML-моделью, определяющей количество людей за столами по видеопотоку.

Функциональность
Отображение текущей посадки:

    Общее количество занятых мест
    Прогресс-бар загруженности с цветовой индикацией
    Цветовая схема столов:
        зелёный — свободно
        жёлтый — частично занято
        красный — стол полностью занят

Анимации и UX:
    Плавное появление столов
    Skeleton-загрузчик
    Выезжающая снизу панель со статистикой

Статистика:
Панель содержит:
    Загруженность по дням недели
    Среднюю загрузку за день
    Пиковое время
    График по часам

Установка и запуск
1. Установите зависимости:
npm install

2. Запустите проект:
npm run dev

Сайт откроется по адресу:
http://localhost:3000
